# BMI 計算器

**第一次上課練習**

## 📌 項目介紹

這是一個使用 **Windows Forms** 開發的 BMI（身體質量指數）計算應用程序。

---

## ✨ 主要功能

### 1. **BMI 計算**
- 公式：`BMI = 體重(公斤) / 身高²(公尺²)`
- 輸入身高（公分）和體重（公斤）即時計算 BMI 值
- 精確到小數點後一位

### 2. **體型分級評估**
根據 BMI 值自動判斷體型分類

| 分類 | BMI 範圍 |
|------|---------|
| 體重過低(輕度) | < 16.5 |
| 體重過低(中度) | 16.5 ~ 18.5 |
| 正常範圍 | 18.5 ~ 25 ✓ |
| 體重過重(輕度) | 25 ~ 27 |
| 體重過重(中度) | 27 ~ 30 |
| 肥胖(輕度) | 30 ~ 35 |
| 肥胖(中度) | 35 ~ 40 |
| 肥胖(重度) | ≥ 40 |

### 3. **預設值設定**
- 默認身高：**170 公分**（男性平均值）
- 默認體重：**70 公斤**（男性平均值）
- 灰色背景標示預設值

### 4. **輸入驗證**
- 身高範圍：1 ~ 300 公分
- 體重範圍：1 ~ 500 公斤
- 智能防呆機制

### 5. **用户交互**
- ✅ 實時計算
- ✅ 色碼反饋
- ✅ 重置功能
- ✅ 友好提示

## 🚀 使用方法

1. **輸入身高和體重** - 灰色背景的預設值會在輸入時自動清除
2. **查看結果** - BMI 值、分類和色碼會實時更新
3. **重置** - 點擊按鈕恢復預設值
