# BMI 計算器

## ?? 項目介紹

這是一個使用 **Windows Forms** 開發的 BMI（身體質量指數）計算應用程序。
作為第一次上課練習項目，涵蓋了基本的 Windows Forms 開發技能。

---

## ? 主要功能

### 1. **BMI 計算**
- 公式：`BMI = 體重(公斤) / 身高?(公尺?)`
- 輸入身高（公分）和體重（公斤）即時計算 BMI 值
- 精確到小數點後一位

### 2. **體型分級評估**
根據 BMI 值自動判斷體型分類：

| 分類 | BMI 範圍 | 顏色 |
|------|---------|------|
| 體重過低(輕度) | < 16.5 | 深藍色 |
| 體重過低(中度) | 16.5 ~ 18.5 | 藍色 |
| 正常範圍 | 18.5 ~ 25 | 綠色 ? |
| 體重過重(輕度) | 25 ~ 27 | 金黃色 |
| 體重過重(中度) | 27 ~ 30 | 橙色 |
| 肥胖(輕度) | 30 ~ 35 | 紅橙色 |
| 肥胖(中度) | 35 ~ 40 | 紅色 |
| 肥胖(重度) | ? 40 | 深紅色 |

### 3. **預設值設定**
- 默認身高：**170 公分**（男性平均值）
- 默認體重：**70 公斤**（男性平均值）
- 灰色背景標示預設值，用?輸入時自動轉為白色

### 4. **輸入驗證**
- 身高範圍：1 ~ 300 公分
- 體重範圍：1 ~ 500 公斤
- 清空輸入框時不觸發錯誤提示
- 只在輸入無效值時才顯示警告

### 5. **用?交互**
- ? 實時計算：輸入時立即更新結果
- ? 色碼反饋：不同健康狀態用不同顏色表示
- ? 重置功能：一鍵恢復預設值
- ? 友好的提示信息：清晰的分類標準說明

## ?? 使用方法

### 1. 輸入身高和體重
- 在「身高(公分)」欄位輸入你的身高
- 在「體重(公斤)」欄位輸入你的體重
- 灰色背景的預設值會在輸入時自動清除

### 2. 查看計算結果
- **BMI 值**：自動顯示計算結果
- **體型分類**：展示對應的健康評估
- **色碼提示**：顏色深淺反映健康程度

### 3. 重置為預設值
- 點擊「重置為預設值」按鈕
- 恢復為男性平均身高和體重
- 輸入框背景轉回灰色
