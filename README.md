# BMI 計算器

## 項目介紹
這是第一次上課練習項目，使用 Windows Forms 開發的 BMI（身體質量指數）計算應用程序。

## 功能特色

### 1. BMI 計算
- **公式**：BMI = 體重(公斤) / 身高?(公尺?)
- 支持公分和公斤單位的輸入
- 實時計算，輸入即時顯示結果

### 2. 詳細的體型分類
提供 8 個分級的健康評估標準：

| 分類 | BMI 範圍 | 顏色指示 |
|------|---------|--------|
| 體重過低(輕度) | BMI < 16.5 | 深藍色 |
| 體重過低(中度) | 16.5 ? BMI < 18.5 | 藍色 |
| 正常範圍 | 18.5 ? BMI < 25 | 綠色 |
| 體重過重(輕度) | 25 ? BMI < 27 | 金黃色 |
| 體重過重(中度) | 27 ? BMI < 30 | 橙色 |
| 肥胖(輕度) | 30 ? BMI < 35 | 紅橙色 |
| 肥胖(中度) | 35 ? BMI < 40 | 紅色 |
| 肥胖(重度) | BMI ? 40 | 深紅色 |

### 3. 預設值設定
- 默認身高：170 公分（男性平均值）
- 默認體重：70 公斤（男性平均值）
- 灰色背景標示預設值，用?輸入時自動轉為白色背景

### 4. 輸入驗證
- **身高驗證**：1-300 公分範圍
- **體重驗證**：1-500 公斤範圍
- 智能防呆機制，只在輸入無效時顯示警告
- 支持清空輸入框而不觸發錯誤

### 5. 用戶友好的 UI
- 明確的色碼提示：不同健康狀態用不同顏色標示
- 清晰的分類標準說明
- 重置按鈕快速恢復預設值

## 使用方法

1. **輸入身高和體重**
   - 在對應的輸入框中輸入數值（單位：公分/公斤）
   - 灰色背景的預設值會在你開始輸入時自動清除

2. **查看計算結果**
   - BMI 值會實時顯示
   - 體型分類和相應的色碼指示會自動更新

3. **重置為預設值**
   - 點擊「重置為預設值」按鈕
   - 恢復到男性平均身高 170 公分、體重 70 公斤

4. **輸入驗證**
   - 若輸入超出有效範圍，會顯示錯誤提示
   - 清空輸入框時不會觸發錯誤
